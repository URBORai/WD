========================================
電子喜帖網站 - 使用說明
========================================
版本：v3.0
最後更新：2025-11-08

📁 檔案結構：
├── index.html          # 主要網頁檔案
├── style.css           # 樣式檔案
├── config.js           # ⭐ 配置檔（修改網站內容請編輯此檔案）
├── 同心同行TJC黎明.mp3  # 背景音樂檔案
├── README.txt          # 本說明檔
└── 規格書.txt          # 技術規格文件

========================================

🎯 如何修改網站內容：

1. 開啟 config.js 檔案
2. 找到您想修改的項目
3. 修改引號內的文字
4. 儲存檔案
5. 重新整理瀏覽器即可看到變更

========================================

📝 必須修改的項目（在 config.js 中）：

✅ 新人資訊：
   - CONFIG.couple.groom = "新郎名字"
   - CONFIG.couple.bride = "新娘名字"

✅ 婚禮日期與時間：
   - CONFIG.wedding.date = "2025年12月25日"
   - CONFIG.wedding.dayOfWeek = "星期四"
   - CONFIG.wedding.time = "晚上 18:00 入席"

✅ 地點地圖：
   - CONFIG.maps.church = "..." (真耶穌教會板橋教會 Google Maps 嵌入網址)
   - CONFIG.maps.restaurant = "..." (天賜良緣 Google Maps 嵌入網址)

✅ 外部連結：
   - CONFIG.links.rsvpForm = "Microsoft 表單連結"
   - CONFIG.links.oldInvitation = "舊版喜帖連結"

========================================

🗺️ 如何取得 Google Maps 嵌入網址：

方法一：使用簡易格式（推薦）
直接使用地址：
   CONFIG.maps.church = "https://www.google.com/maps?q=新北市板橋區大觀路2段2號&output=embed"
   CONFIG.maps.restaurant = "https://www.google.com/maps?q=242新北市新莊區中正路542號&output=embed"

方法二：使用 Google Maps 嵌入網址（更精確）
1. 前往 Google Maps (https://maps.google.com)
2. 搜尋您的地點
3. 點擊「分享」按鈕
4. 選擇「嵌入地圖」
5. 點擊「複製 HTML」
6. 從複製的代碼中找到 src="..." 部分
7. 將引號內的網址複製到 CONFIG.maps.church 或 CONFIG.maps.restaurant

範例：
   <iframe src="這裡的網址就是您要的">
   複製：https://www.google.com/maps/embed?pb=...

========================================

🎵 背景音樂設定（在 config.js 中）：

- CONFIG.music.filename = "同心同行TJC黎明.mp3"
  （如果更換音樂檔案，請修改此處的檔案名稱）

- CONFIG.music.fadeInDuration = 2000
  （音樂淡入時間，2000 = 2秒）

- CONFIG.music.maxVolume = 0.7
  （最大音量，0.7 = 70%，範圍 0.0-1.0）

========================================

🎨 進階設定：

📌 文案內容：
   所有顯示的文字都可以在 CONFIG.content 中修改

📌 彈窗設定：
   歡迎彈窗的文字可以在 CONFIG.welcomePopup 中修改

📌 動畫設定：
   滾動動畫的參數可以在 CONFIG.aos 中調整

📌 配色：
   主要顏色記錄在 CONFIG.colors 中
   ⚠️ 注意：修改配色需要同時修改 style.css 檔案

========================================

📷 圖片設定：

本網站包含以下圖片區塊：
1. 關於我們區塊：couple-photo.jpg (新人合照)

如何替換圖片：
1. 準備您的照片檔案（建議使用 JPG 格式，檔案 < 500KB）
2. 將照片重新命名為 couple-photo.jpg
3. 將照片放入專案資料夾（與 index.html 同一層）
4. 重新整理網頁即可看到新照片

💡 圖片優化建議：
- 使用線上工具壓縮圖片（如 TinyPNG）
- 建議圖片尺寸：800x600px 或 1200x800px
- 保持良好的畫質與檔案大小平衡

📋 網站內容區塊說明：

本網站包含以下區塊（依序由上至下）：

1️⃣ 首頁 (Hero) - 新人名字與婚禮日期
2️⃣ 關於我們 (About) - 新人故事簡介與婚禮資訊
3️⃣ 婚禮進程 (Timeline) - 當日時間軸（入場、儀式、晚宴、敬酒等）
4️⃣ 回覆出席 (RSVP) - 兩張連結卡片（表單 + 舊版喜帖）
5️⃣ 地點 (Venue) - 兩個定位按鈕 + Google Maps 互動地圖
   - 📍 真耶穌教會板橋教會
   - 📍 天賜良緣
   - 點擊按鈕可切換地圖顯示的地點

⚠️ 注意：如需修改婚禮進程的詳細內容，
   需要直接編輯 index.html 檔案中對應的區塊。

🌐 如何在本地測試：

方法一：使用 VS Code + Live Server
1. 用 VS Code 開啟專案資料夾
2. 安裝 "Live Server" 擴充套件
3. 右鍵點擊 index.html
4. 選擇 "Open with Live Server"

方法二：直接開啟
1. 直接雙擊 index.html
2. 會在瀏覽器中開啟
3. ⚠️ 某些功能可能因瀏覽器安全限制無法正常運作

📤 部署到 Azure：

1. 將所有檔案上傳到 GitHub 儲存庫
2. 登入 Azure Portal (https://portal.azure.com)
3. 建立新的 "Static Web Apps"
4. 連結到您的 GitHub 儲存庫
5. Azure 會自動部署
6. 取得網址後，記得測試所有功能是否正常

其他部署選項：
- GitHub Pages（免費，適合個人專案）
- Netlify（免費，提供自動部署）
- Vercel（免費，速度快）

⚠️ 注意：使用 Google Maps 嵌入式地圖通常不需要 API Key

❓ 常見問題：

Q: 為什麼音樂沒有播放？
A: 1. 確認音樂檔案名稱與 config.js 中設定的相符
   2. 確認音樂檔案與 index.html 在同一個資料夾
   3. 瀏覽器可能阻擋自動播放，請確認已點擊彈窗的「開始體驗」按鈕

Q: 如何修改顏色？
A: 需要編輯 style.css 檔案，搜尋顏色代碼（例如 #d4a574）並替換

Q: 地圖沒有顯示？
A: 確認 Google Maps 嵌入網址是否正確，並檢查網址是否有權限限制

Q: 修改 config.js 後沒有變化？
A: 按 Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac) 強制重新整理

Q: 如何修改地點按鈕的名稱？
A: 需要直接編輯 index.html，找到地點區塊的按鈕部分修改文字

Q: 地點按鈕點擊後地圖沒有切換？
A: 1. 確認 config.js 中的 maps.church 和 maps.restaurant 網址正確
   2. 打開瀏覽器開發者工具 (F12) 查看是否有錯誤訊息
   3. 確認網址格式正確（應包含 &output=embed 或使用 Google Maps embed URL）

📧 技術支援：

如有任何問題，請檢查：
1. 瀏覽器的開發者工具 (F12) 中是否有錯誤訊息
2. 檔案名稱是否正確
3. config.js 中的語法是否正確（引號、逗號等）
4. 所有檔案是否都在同一個資料夾內
5. 圖片檔案是否存在且命名正確

💡 測試建議：
- 在不同瀏覽器測試（Chrome, Firefox, Safari, Edge）
- 在不同裝置測試（電腦、平板、手機）
- 測試所有連結是否正確運作
- 確認音樂播放正常
