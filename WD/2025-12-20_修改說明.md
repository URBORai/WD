## 🎨 功能特點

### 1. 三種切換方式
- ✅ 點擊「新郎」/「新娘」按鈕切換
- ✅ 點擊左側箭頭（‹）切換
- ✅ 點擊右側箭頭（›）切換

### 2. 流暢動畫效果
- 淡入淡出效果（opacity）
- 滑動效果（transform）
- 使用 visibility 控制元素可見性
- 50ms 延遲確保動畫流暢

### 3. 響應式設計
- 桌面版：左右排列（照片在左、資訊在右）
- 手機版：上下排列（照片在上、資訊在下）

### 4. 互動反饋
- 按鈕懸浮效果
- 照片懸浮放大效果
- 卡片陰影效果

## 🔧 如何自訂內容

### 修改新郎/新娘資料
編輯 `config.js` 文件：

```javascript
groomProfile: {
    name: "您的名字",              // 修改名字
    title: "新郎",                  // 修改頭銜
    photo: "../圖片/您的照片.jpg",  // 修改照片路徑
    description: "您的個人介紹...", // 修改描述（可用 \n 換行）
    hobbies: ["興趣1", "興趣2"],    // 修改興趣愛好（陣列）
    quote: "您的座右銘"             // 修改引言
}
```

### 修改樣式顏色
如果要改變主題色，在 `style.css` 中搜尋 `#d4a574`（金色）並替換為您喜歡的顏色。

### 調整動畫速度
在 `style.css` 第 585 行和第 595 行：
```css
transition: opacity 0.5s ease, transform 0.5s ease, ...
```
將 `0.5s` 改為您想要的秒數（例如 `0.3s` 更快，`0.8s` 更慢）。

## ⚠️ 注意事項

1. **照片路徑**
   - 確保 `photo` 欄位中的照片路徑正確
   - 預設照片放在 `../圖片/` 資料夾中

2. **興趣愛好數量**
   - 建議不超過 5 個，避免排版過於擁擠
   - 每個興趣名稱建議 2-4 個字

3. **個人描述長度**
   - 建議 3-5 行，保持簡潔
   - 使用 `\n` 換行

4. **瀏覽器相容性**
   - 支援現代瀏覽器（Chrome、Firefox、Safari、Edge）
   - 建議使用最新版本瀏覽器以獲得最佳體驗

## 🐛 問題修復記錄

### 問題 1：切換功能不工作
**原因：** CSS 中 `.active` 狀態的 `position` 從 `absolute` 變為 `relative` 導致布局錯亂

**解決方案：**
- 兩張卡片都保持 `position: absolute`
- 使用 `visibility` 和 `z-index` 控制顯示層級
- 優化 transition 時序

### 問題 2：動畫不流暢
**原因：** AOS 動畫與自定義切換動畫衝突

**解決方案：**
- 移除卡片上的 AOS 屬性（`data-aos`）
- 在 JavaScript 切換時加入 50ms 延遲
- 使用 `setTimeout` 確保過渡效果
